Vueのアプリをテストファーストで書いてみます。

## 雛形のアプリを作る
新しいディレクトリにVueのアプリを作ります。
手元のXubuntu 24.01には`yarn`パッケージとしてyarn 1.22.15が入っています。

```shell
$ mkdir clean-url
$ cd clean-url
$ yarn init
$ yarn add -D @vue/cli
$ yarn run vue create .
```

`package.json`を眺めると、`yarn serve`で開発用のサーバを起動できそうです。

```json
{
  "scripts": {
    "serve": "vue-cli-service serve"
  }
}
```

サーバが起動しました。念のためブラウザで閲覧できるのを確認します。

```shell

```

## ユニットテストを書いて実行する
今回は、入力されたURLを加工するアプリを書いてみます。

まず、URLをパーズするヘルパ関数を書きます。
`src/helpers.js`に、とりあえず最小の機能を持った下記の関数を定義します。

```js
export function parseUrl(url) {
  return URL.parse(url)
}
```

Vueによる[Testing](https://vuejs.org/guide/scaling-up/testing.html#recommendation)ガイドによると、テストフレームワークには[Vitest](https://vitest.dev/)が推奨のようです。インストールします。

```shell
$ yarn add -D vitest
```

[Getting Started](https://vitest.dev/guide/#writing-tests)ガイドに従って、
`package.json`に`test`スクリプトを追加します。

```json
{
  "scripts": {
    "test": "vitest"
  }
}
```

さらに、`src/helpers.spec.js`にユニットテストを書きます。

```js
import { describe, expect, test } from 'vitest'
import { parseUrl } from './helpers'

describe('parseUrl', () => {
  test('parses a minimal URL', () => {
    const parsed = parseUrl('https://example.com')
    const expected = { host: 'example.com', protocol: 'https:' }
    Object.entries(expected).forEach(([prop, goal]) => {
      expect(parsed[prop]).toBe(goal)
    })
  })
})
```

ユニットテストを実行します。
